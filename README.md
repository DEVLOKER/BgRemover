# BgRemover
Python application for removing image background, developed based on [ModNet](https://github.com/ZHKKKe/MODNet){:target="_blank" rel="noopener"}.
## Used Tools
`Python`, `Flask`.
## How to install?
### backend
- `pip install flask`
- install [ModNet](https://github.com/ZHKKKe/MODNet)
## How to run?
- `flask run`
