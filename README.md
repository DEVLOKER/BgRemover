# BgRemover
Python application for removing image & video background, developed based on [ModNet](https://github.com/ZHKKKe/MODNet).
## Used Tools
`Python`, `Flask`, `openCV`.
## How to install?
### backend
- `pip install flask`
- install [ModNet](https://github.com/ZHKKKe/MODNet)
## How to run?
- `flask run`
